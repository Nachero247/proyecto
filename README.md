# 🏋️ GYM MANAGER
**Gym Manager** es una aplicación de escritorio desarrollada en **Java con NetBeans** para la gestión de un gimnasio. Permite administrar usuarios del sistema, socios, pagos, planes de entrenamiento y asistencias de forma centralizada y sencilla.

---

## ⚙️ Funcionalidades principales
- 👤 Gestión de usuarios del sistema (creación y administración)
- 🧍 Gestión de socios del gimnasio
- 💳 Control de pagos y cuotas
- 📅 Gestión de planes de entrenamiento
- 🏃 Registro de asistencias de socios
- 🧭 Menú principal con acceso a todos los módulos
- 📖 Manual de usuario integrado en el menú principal

---

## 🖥️ Interfaz
La aplicación cuenta con un menú principal con botones, desde el cual se accede a todos los apartados de forma rápida e intuitiva.

---

## 🛠️ Tecnologías utilizadas
- Java
- NetBeans IDE
- MySQL

---

## 📥 Pasos de instalación
1. Descargar o clonar el repositorio:
2. Abrir NetBeans
3. Selecccionar File -> Open Project
4. Seleccionar la carpeta del proyecto
5. Esperar a que NetBeans cargue el proyecto y las dependencias
6. Verificar que las librerías estén añadidas:
- Click derecho en el proyecto → Properties
- Ir a Libraries
- Comprobar que todas las librerías de la carpeta 'lib' están añadidas correctamente
7. Ejecutar el proyecto:
- Ejecutar archivo JFrameLogin.java
  
---

## 📖 Manual de usuario
La aplicación incluye un manual accesible desde el menú principal, donde se explican todas las funciones disponibles.
