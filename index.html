GYM MANAGER · POTENCIA TU GIMNASIO

Aplicación desarrollada en Java con NetBeans y base de datos MySQL para gestionar un gimnasio de forma centralizada. Administra usuarios, socios, pagos, planes de entrenamiento y asistencias con total sencillez.

☕ Java / NetBeans 🐬 MySQL 📊 Informes XML/JSON 🔐 Roles + Permisos

⚙️ Funcionalidades · Todo lo que necesitas

Una plataforma completa que cubre cada aspecto de la gestión de tu gimnasio desde un solo lugar.

👥

Gestión de Socios

Crea, edita y controla el estado de cada socio: alta, baja o suspendido. Almacena DNI, teléfono, correo electrónico y toda la información relevante de forma centralizada.

💳

Control de Pagos

Registra pagos con estado: Pagado, Pendiente o Vencido. Filtra por DNI, estado o fecha. Imprime informes detallados de pagos pendientes o por socio en segundos.

🏋️

Planes de Suscripción

Crea planes personalizados con precio y duración: 1 mes, 3 meses, 6 meses, anual o los que necesites. Asigna planes a cada socio y controla sus vencimientos automáticamente.

📋

Registro de Asistencias

Guarda la fecha exacta de cada visita al gimnasio por socio. Genera informes de asistencia y analiza la frecuencia de uso para tomar mejores decisiones.

📊

Informes Imprimibles

Imprime informes de socios, pagos y planes filtrando por cualquier criterio. Exporta e importa datos desde archivos XML y JSON para máxima flexibilidad.

🔐

Usuarios del Sistema

Administra los accesos con roles diferenciados: Administrador, Entrenador y Profesor. Cada rol tiene permisos específicos sobre los módulos de la aplicación.

Manual de Usuario · Guía paso a paso

Todo lo que necesitas saber para usar GymManager. Selecciona el módulo sobre el que quieres obtener ayuda.

🔑 Inicio de sesión

La pantalla de Login es el punto de entrada al sistema. Solo los usuarios registrados por el Administrador pueden acceder.

  1. Introduzca su nombre de usuario en el campo Usuario.
  2. Introduzca su contraseña en el campo Contraseña.
  3. Pulse el botón Iniciar Sesión.
  4. Si las credenciales son correctas, accederá al Menú Principal con las opciones correspondientes a su rol.
⚠️ Si las credenciales son incorrectas el sistema mostrará un error. Compruebe mayúsculas y espacios. Si no recuerda su contraseña, contacte con el Administrador.

🏠 Menú Principal

Tras el login verá el Menú Principal con los módulos habilitados para su rol. Use Cerrar Sesión para volver a la pantalla de Login de forma segura.

👥 Gestión de Socios

Muestra la tabla completa de socios registrados. Permite filtrar por nombre, DNI o estado para localizar cualquier miembro rápidamente.

Acciones disponibles: Nuevo Socio, Editar Socio, Eliminar Socio (solo Administrador) y Buscar / Filtrar.

Campos visibles en la tabla: ID · Nombre · Apellidos · DNI · Teléfono · Correo · Fecha Alta · Estado

➕ Dar de alta un nuevo socio

Pulse Nuevo Socio para abrir el formulario de alta. Rellene los siguientes campos:

CampoDescripciónObligatorio
NombreNombre del socio (máx. 30 car.)*
Primer ApellidoPrimer apellido (máx. 30 car.)*
Segundo ApellidoSegundo apellido (máx. 30 car.)
DNIIdentificador único (máx. 20 car.)*
TeléfonoNúmero de contacto (máx. 30 car.)*
Correo electrónicoEmail válido (máx. 60 car.)*
Fecha de AltaFormato AAAA-MM-DD*
EstadoAlta, Baja o Suspendido*
⚠️ El DNI debe ser único. Si ya existe un socio con ese DNI el sistema mostrará un error.

✏️ Editar un socio

Seleccione el socio en la tabla y pulse Editar. El formulario cargará los datos actuales. Modifique lo que necesite y pulse Guardar. El ID del socio no es editable.

Puede cambiar el estado entre Alta, Baja y Suspendido desde el selector de estado del mismo formulario.

✔ Los cambios solo se aplican al pulsar Guardar. Cancelar cierra sin guardar.

🧑‍💼 Gestión de Usuarios

Módulo exclusivo del Administrador. Permite crear, editar y eliminar las cuentas de acceso al sistema. Si otro rol intenta acceder será redirigido al Menú Principal.

Campos visibles: ID · Usuario · Rol · Plan asociado. La contraseña no se muestra en la tabla por seguridad.

➕ Crear un nuevo usuario

CampoDescripciónObligatorio
Nombre de usuarioIdentificador único de acceso (máx. 30 car., sin espacios)*
ContraseñaClave de acceso (máx. 50 car.)*
RolAdministrador, Entrenador o Profesor*
Plan asociadoPlan de membresía vinculado (opcional)
⚠️ El nombre de usuario debe ser único. No se permiten espacios ni caracteres especiales.

✏️ Editar un usuario

Seleccione el usuario y pulse Editar. Puede modificar el nombre, contraseña, rol y plan asociado.

Si deja el campo Contraseña vacío, la contraseña actual no se modificará. El cambio de rol tiene efecto en el próximo inicio de sesión del usuario.

💳 Gestión de Pagos

Permite registrar y consultar los pagos de los socios. Filtre por socio o estado para localizar registros rápidamente.

Campos de la tabla: ID_Pago · Socio · Fecha de Pago · Importe (€) · Estado

💡 Estados del pago

✅ Pagado

  • El pago ha sido recibido correctamente.

⏳ Pendiente

  • Registrado pero aún no cobrado.

🔴 Vencido

  • Superó la fecha límite sin abonarse.

➕ Registrar un nuevo pago

  1. Pulse Nuevo Pago en la pantalla de Gestión de Pagos.
  2. Seleccione el socio de la lista desplegable.
  3. Introduzca el importe y la fecha.
  4. Seleccione el estado inicial (Pagado o Pendiente).
  5. Pulse Guardar para registrar el pago.
🔒 Solo el Administrador puede eliminar pagos. El Profesor únicamente puede consultar el listado.

🏋️ Gestión de Planes

Administra los planes de membresía asociados a los socios: mensual, trimestral, semestral, anual o cualquier duración personalizada.

Campos de la tabla: ID_Plan · Socio · Descripción · Importe (€) · Fecha Inicio · Fecha Fin

➕ Asignar un nuevo plan

  1. Pulse Nuevo Plan.
  2. Seleccione el socio al que se asignará el plan.
  3. Escriba una descripción (p. ej. "Plan mensual básico").
  4. Introduzca el importe en euros correspondiente al plan.
  5. Indique la Fecha Inicio y la Fecha Fin del plan.
  6. Pulse Guardar.

📅 Vigencia del plan

Un plan está vigente cuando la fecha actual se encuentra entre la Fecha Inicio y la Fecha Fin.

⚠️ El sistema no bloquea el acceso automáticamente al vencer un plan. Debe gestionarse manualmente cambiando el estado del socio a Baja o Suspendido.

El Administrador tiene acceso completo. Entrenador y Profesor solo pueden consultar.

📅 Gestión de Asistencias

Permite registrar y consultar las entradas de los socios al gimnasio, controlando fecha y hora de cada visita.

Campos de la tabla: ID_Asistencia · Socio · Fecha de Asistencia · Hora de Entrada

➕ Registrar una asistencia

  1. Pulse Registrar Asistencia.
  2. Seleccione el socio de la lista desplegable.
  3. La fecha y hora se rellenan automáticamente con el momento actual. Pueden modificarse manualmente si fuera necesario.
  4. Pulse Guardar.

🔍 Filtrar por socio o fecha

Use los filtros disponibles para consultar el historial de asistencias de un socio concreto o de un periodo determinado. Ideal para generar informes de frecuencia de uso.

🔒 El Administrador y el Entrenador pueden registrar y gestionar asistencias. El Profesor no tiene acceso a este módulo.
