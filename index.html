GYM MANAGER · POTENCIA TU GIMNASIO

Aplicación desarrollada en Java con NetBeans y base de datos MySQL para gestionar un gimnasio de forma centralizada. Administra usuarios, socios, pagos, planes de entrenamiento y asistencias con total sencillez.

☕ Java / NetBeans 🐬 MySQL 📊 Informes XML/JSON 🔐 Roles + Permisos

⚙️ Funcionalidades · Todo lo que necesitas

Una plataforma completa que cubre cada aspecto de la gestión de tu gimnasio desde un solo lugar.

👥

Gestión de Socios

Crea, edita y controla el estado de cada socio: alta, baja o suspendido. Almacena DNI, teléfono, correo electrónico y toda la información relevante de forma centralizada.

💳

Control de Pagos

Registra pagos con estado: Pagado, Pendiente o Vencido. Filtra por DNI, estado o fecha. Imprime informes detallados de pagos pendientes o por socio en segundos.

🏋️

Planes de Suscripción

Crea planes personalizados con precio y duración: 1 mes, 3 meses, 6 meses, anual o los que necesites. Asigna planes a cada socio y controla sus vencimientos automáticamente.

📋

Registro de Asistencias

Guarda la fecha exacta de cada visita al gimnasio por socio. Genera informes de asistencia y analiza la frecuencia de uso para tomar mejores decisiones.

📊

Informes Imprimibles

Imprime informes de socios, pagos y planes filtrando por cualquier criterio. Exporta e importa datos desde archivos XML y JSON para máxima flexibilidad.

🔐

Usuarios del Sistema

Administra los accesos con roles diferenciados: Administrador, Entrenador y Profesor. Cada rol tiene permisos específicos sobre los módulos de la aplicación.

🛡️ Roles y Accesos · Control de permisos

Tres roles de usuario con accesos diferenciados para mantener la seguridad y organización de tu equipo.

🛡️ Administrador

  • ✅ Gestión completa de socios
  • ✅ Control de pagos y cuotas
  • ✅ Gestión de planes
  • ✅ Administración de usuarios
  • ✅ Todos los informes / Importar-Exportar

💪 Entrenador

  • ✅ Ver y editar socios
  • ✅ Registro de asistencias
  • ✅ Consultar planes / Informes asistencia
  • ❌ Sin acceso a pagos
  • ❌ Sin gestión de usuarios

📚 Profesor

  • ✅ Consulta de socios
  • ✅ Registro de asistencias
  • ✅ Ver planes asignados
  • ❌ Sin edición de datos
  • ❌ Sin acceso a pagos / gestión usuarios
