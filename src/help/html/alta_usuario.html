Alta de Usuario (JDialog)

Permite crear una nueva cuenta de acceso al sistema. Solo accesible para el rol Administrador.

Campos del formulario

CampoDescripciónObligatorio
Nombre de usuarioIdentificador único de acceso (máx. 30 caracteres)*
ContraseñaClave de acceso (máx. 50 caracteres)*
RolSeleccione: Administrador, Entrenador o Profesor*
Plan asociadoPlan de membresía vinculado (opcional)
⚠️ El nombre de usuario debe ser único. No se permiten espacios ni caracteres especiales.

Botones

Guardar – Crea la cuenta de usuario.
Cancelar – Cierra sin guardar.
