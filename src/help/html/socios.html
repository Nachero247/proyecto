Gestión de Socios

Esta ventana permite administrar todos los socios del gimnasio. Muestra una tabla con todos los socios registrados y sus datos principales.

Acciones disponibles

➕ Nuevo Socio – Abre el diálogo de Alta de Socio para registrar un nuevo miembro.
✏️ Editar Socio – Seleccione un socio de la tabla y pulse Editar para modificar sus datos.
🗑️ Eliminar Socio – Elimina el socio seleccionado (solo Administrador). Esta acción es irreversible.
🔍 Buscar / Filtrar – Filtre por nombre, DNI o estado para localizar socios rápidamente.

Campos de la tabla

ID · Nombre · Apellidos · DNI · Teléfono · Correo · Fecha Alta · Estado

Estados posibles de un socio

Un socio puede estar en uno de estos tres estados: Alta (activo), Baja (dado de baja) o Suspendido (acceso temporalmente bloqueado).
