Editar Usuario (JDialog)

Permite modificar los datos de una cuenta de usuario existente. Solo el Administrador puede usar esta función.

Funcionamiento

Al abrirse, el formulario carga los datos actuales del usuario seleccionado. Puede modificar el nombre de usuario, la contraseña, el rol y el plan asociado.

Cambio de contraseña

Si desea cambiar la contraseña, escriba la nueva en el campo Contraseña. Si deja el campo vacío, la contraseña actual no se modificará.

Cambio de rol

El cambio de rol tiene efecto inmediato en el próximo inicio de sesión del usuario. Asegúrese de comunicar al usuario cualquier cambio de rol que afecte a sus permisos.

✔ Los cambios se guardan en la base de datos únicamente al pulsar Guardar.

Botones

Guardar – Aplica los cambios.
Cancelar – Descarta los cambios y cierra el diálogo.
