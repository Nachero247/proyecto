Gestión de Usuarios

Módulo exclusivo del Administrador. Permite administrar las cuentas de acceso al sistema (usuarios, contraseñas y roles).

Acciones disponibles

➕ Nuevo Usuario – Abre el diálogo de Alta de Usuario.
✏️ Editar Usuario – Modifica los datos del usuario seleccionado.
🗑️ Eliminar Usuario – Elimina la cuenta del usuario seleccionado.

Campos de la tabla

ID · Usuario · Rol · Plan asociado

Por seguridad, la contraseña no se muestra en la tabla de gestión.

Relación con Socios y Planes

Un usuario puede tener asociado un Plan (campo Plan_ID), que enlaza con los planes de membresía de los socios. Esta relación es opcional.

🔒 Solo el Administrador puede acceder a este módulo. Si otro rol intenta acceder, el sistema lo redirigirá al Menú Principal.
