Roles y Permisos

El sistema de gimnasio gestiona el acceso mediante tres roles diferenciados. El rol se asigna al crear o editar un usuario desde Gestión de Usuarios.

🛡️ Administrador

Tiene acceso completo a todos los módulos: puede crear, editar y eliminar socios, usuarios, pagos, planes y asistencias. Es el único rol que puede gestionar cuentas de usuario.

🏋️ Entrenador

Puede consultar socios y planes, y gestionar asistencias (registrar entradas). No tiene acceso a pagos ni a la gestión de usuarios.

📚 Profesor

Puede consultar planes y pagos, pero no puede modificar datos de socios, usuarios ni asistencias.

Nota: Si necesita cambiar el rol de un usuario, contacte con el Administrador del sistema.
