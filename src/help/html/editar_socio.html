Editar Socio (JDialog)

Este diálogo permite modificar los datos de un socio ya registrado. Se abre seleccionando un socio en la tabla de Gestión de Socios y pulsando Editar.

Funcionamiento

Al abrirse, el formulario carga automáticamente todos los datos actuales del socio seleccionado. Modifique únicamente los campos que necesite cambiar y pulse Guardar para aplicar los cambios.

Campos modificables

Todos los campos del socio son editables: Nombre, Apellidos, DNI, Teléfono, Correo electrónico, Fecha de Alta y Estado. El ID del socio no es editable, ya que es el identificador interno del sistema.

Cambio de estado

Puede cambiar el estado del socio entre Alta, Baja y Suspendido desde este mismo formulario usando el selector de estado.

✔ Los cambios solo se aplican a la base de datos cuando pulsa Guardar. Si cierra el diálogo con Cancelar, no se guardará ninguna modificación.

Botones

Guardar – Aplica los cambios y cierra el diálogo.
Cancelar – Descarta los cambios y cierra el diálogo.
